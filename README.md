## Featured Projects

### ♿ Incluyete Estepon@ — *in development*

Tourism accessibility platform built in collaboration with **Estepona City Council**, focused on improving inclusive tourism for people with disabilities.

- REST API with **Spring Boot** + frontend in **React**
- Interactive accessibility maps integrated with local services
- Live preview: [incluyete.duckdns.org](https://incluyete.duckdns.org/) · Organization: [AsociacionIncluyete](https://github.com/AsociacionIncluyete)

### Smart ERP System — *in progress*

Custom ERP showcasing full-stack and DevOps capabilities with microservices architecture and self-hosted CI/CD.

- Docker + Kubernetes deployments on own infrastructure
- Modules: inventory, user management, reporting
- Demo: [webportfolio.app](https://webportfolio.app/)

### Self-Hosted Server Infrastructure

Production-grade homelab on **Orange Pi 5 Plus** — fully containerized services with Traefik (auto SSL), Nginx, Apache and health monitoring dashboards.

- Docs: [server-doc.duckdns.org](https://server-doc.duckdns.org/) · Repo: [orange-pi5plus-server-docs](https://github.com/eChrls/orange-pi5plus-server-docs)
